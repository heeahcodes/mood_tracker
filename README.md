# mood_tracker

A simple Python program that lets you log your mood with the current date, time, and day.  
Your entries are saved into a CSV file for later analysis.

---

## Features
- Log moods using numeric codes.
- Automatically records the date, time, and day of the week
- Saves entries to a CSV file (`mood_log.csv`)
- Easy to extend for analysis or visualization hereafter

---

### How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/mood_tracker.git
   cd mood_tracker
2. Run the script:
   python mood_tracker.py
3. Enter mood (1-8) when prompted : 5
   "Your mood has been recorded. Take care."
